How to Install

Three simple steps to get started.

1

Install VLC Media Player

Match Watcher uses the VLC engine for playback. Download and install it from videolan.org or run:

brew install --cask vlc
2

Download and Open the DMG

Click the download button above. Open the MatchWatcher.dmg file and drag the app into your Applications folder.

3

Bypass Gatekeeper (First Launch Only)

Because Match Watcher is not signed with an Apple Developer certificate, macOS will block it on first launch. See the instructions below to bypass this.

⚠️ macOS Gatekeeper — Cannot be opened

Apple's Gatekeeper blocks apps from unidentified developers by default. This is normal for open-source apps distributed outside the Mac App Store. Here are three ways to fix it:

Option A — Right-click method (easiest):

Right-click (or Control-click) on Match Watcher.app in your Applications folder and choose "Open" from the menu. Click "Open" again in the dialog. You only need to do this once.

Option B — Terminal command:

If the right-click method doesn't work, open Terminal and run:

xattr -cr /Applications/Match\ Watcher.app

Then try launching the app normally.

Option C — System Settings:

After a failed launch, go to System Settings → Privacy & Security. Scroll down and you'll see a message saying Match Watcher was blocked. Click "Open Anyway".
